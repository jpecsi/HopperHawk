Welcome!

This is the HopperHawk Configuration Tool. This web interface was designed to connect, configure, and monitor a HopperHawk device. Please note, this is intended to be a beta solution. The final product will have a mobile app designed for iOS and Android to provide a much better/integrated experience. If you are accessing this site, chances are you are helping to beta test the HopperHawk product...and for that I say thank you :) Below are some tips/instructions for how to use this tool.


Connectivity

Wi-Fi
HopperHawk is designed to connect to your home netork wirelessly. The first time you setup HopperHawk it will broadcast it's own network for you to connect to. If for some reason your wifi is unavailable or HopperHawk fails to connect when it turns on...it will always default to broadcasting it's own network. Unless there is an actual system failure you will always be able to connect!

SSID: HopperHawk-XXXX
Password: None

Once connected to the HopperHawk network, you will be able to reach it using either the IP address 192.168.4.1 or hostname hopperhawk.local
